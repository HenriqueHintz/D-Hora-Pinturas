## 📌 Sobre o Projeto | About
**PT:** O D'Hora Pinturas é uma plataforma web desenvolvida para posicionar a empresa de pinturas e revestimentos como uma parceira de alto padrão para concessonárias, shoppings, varejo e projetos arquitetônicos. O sistema entrega uma experiência sensorial rica — apelidada de essência NOR — através de componentes de alta performance, menus em tela cheia (overlay), animações aceleradas por GPU, crossfading interativos no Hero section via troca dinâmica de texturas e um UI system sofisticado feito do zero. 

**EN:** D'Hora Pinturas is a web platform designed to position a painting and coating company as a premium B2B partner for retail stores, malls, dealerships, and architectural projects. The system delivers a rich sensory experience through high-performance components, full-screen overlay menus, GPU-accelerated animations, dynamic texture crossfading in the hero section, and a sophisticated, custom-built minimalist UI system.

## ⚡ Funcionalidades | Features
- **Dynamic Hero Style Selector:** Troca em tempo real da identidade visual da página inicial baseada no estilo de revestimento escolhido (Granizo, Quartzo, Cerâmica) com animações CSS customizadas.
- **Full-Screen Mobile Menu:** Navegação bloqueada para rolagem interativa com locking scroll overlay perfeitamente em aparelhos menores.
- **Custom High-Performance Video Player:** Componente de player otimizado focando na qualidade (sem botões genéricos nativos), expansão em tela cheia transparente.
- **Glassmorphism Header:** Cabeçalho inteligente e responsivo com efeito translúcido (backdrop-blur-xl).
- **Responsive Architecture (Mobile First):** Ajustes completos de layout e flexbox garantem acessibilidade e UX unificadas tanto no desktop quanto celular.

## ⚙️ Instalação | Setup

1. **Clone do Repositório**
```bash
git clone https://github.com/HenriqueHintz/D-Hora-Pinturas.git
cd D-Hora-Pinturas
```

2. **Instalação das Dependências**
```bash
npm install
```

3. **Iniciando a Aplicação**
```bash
npm run dev
```

Acesse em: `http://localhost:3000`
